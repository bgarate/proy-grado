Enumeration Type Documentation

enum eARDISCOVERY_CONNECTION_STATE

State of a connection This is used for BLE products.

Enumerator
ARDISCOVERY_CONNECTION_STATE_UNKNOWN 

Connection state unknown

ARDISCOVERY_CONNECTION_STATE_AVAILABLE 

Connection is available, product is theoretically connectable

ARDISCOVERY_CONNECTION_STATE_CONNECTED_TO_OTHER 

Product is connected to a controller. This controller is not one of the listed in the following enum values

ARDISCOVERY_CONNECTION_STATE_CONNECTED_TO_TINOS 

Product is connected to a Tinos controller

ARDISCOVERY_CONNECTION_STATE_MAX 

Max value, no meaning, should be considered like Unknown
