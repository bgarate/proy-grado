Parameters

Returns

typedef eARNETWORKAL_MANAGER_RETURN(* ARNETWORKAL_Manager_Receive_t) (ARNETWORKAL_Manager_t *manager)

Read a bunch of frames from the network This function will read a bunch of data from the network, so these datas can be fetched using ARNETWORKAL_Manager_PopFrame_t. On implementations where ARNETWORKAL_Manager_PopFrame_t does an actual network read, this function only waits for data to be available, but does not do an actual read. In any cases, this function should be used by the receiving thread to wait for new network data.

Parameters
managerThe manager which should read from the network
Returns
A callback return code

Enumeration Type Documentation

enum eARNETWORKAL_MANAGER_RETURN

Transfert functions return codes.

Enumerator
ARNETWORKAL_MANAGER_RETURN_DEFAULT 

Default return value (no error)

ARNETWORKAL_MANAGER_RETURN_BUFFER_FULL 

Impossible to push a frame : network buffer is full

ARNETWORKAL_MANAGER_RETURN_BUFFER_EMPTY 

Impossible to pop a frame, no frame in the buffer

ARNETWORKAL_MANAGER_RETURN_BAD_FRAME 

Impossible to pop a frame, frame is corrupted

ARNETWORKAL_MANAGER_RETURN_NO_DATA_AVAILABLE 

Impossible to read data from the network, no data available

ARNETWORKAL_MANAGER_RETURN_BAD_PARAMETERS 

Parameters given to the callback were not good

ARNETWORKAL_MANAGER_RETURN_NETWORK_ERROR 

Network error while reading or sending data
