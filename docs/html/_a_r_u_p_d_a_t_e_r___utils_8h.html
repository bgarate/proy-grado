Extract a U_UNIFXILE regular file from a PLF file.

Example: a U_UNIXFILE section (regular file) with path "data/foo/config.txt" should be extracted by providing the last path component ("config.txt") as parameter .

Parameters
[in]plfFileName: PLF file path
[in]outFolder: Directory path in which the file will be extracted
[in]unixFileName: The name of the file we want to extract; also the name of the file written in outFolder
Returns
ARUPDATER_OK if operation went well, the description of the error otherwise
