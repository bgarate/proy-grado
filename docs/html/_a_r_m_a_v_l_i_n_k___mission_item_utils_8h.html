float  verticalRotationSpeed 
)

Fill a panorama mission item with the given params and the default params.

This item will start a move by the drone or its camera on the yaw/pan axis and on the tilt when it will be read by the drone Note that, the first vertical angle will be applied relatively to the physical center of the camera.

Parameters
[out]missionItem: pointer on the mission item to fill
[in]horizontalAngle: the horizontal relative angle (deg)
[in]verticalAngle: the vertical relative angle (deg) (negative angle moves the camera down).
[in]horizontalRotationSpeed: the desired horizontal rotation speed (m/s)
[in]verticalRotationSpeed: the desired vertical rotation speed (m/s)
Returns
ARMAVLINK_OK if operation went well, the enum description of the error otherwise

[in]altitude: the altitude of the mission item
[in]command: the command of the mission item

[in]current: false:0, true:1
[in]autocontinue: autocontinue to next wp
Returns
