Read atom data from atom into a self alloced array This function get the atom data from a video and convert it to the latest version This function alloc (using vp_os_calloc) the return pointer. Application MUST handle the free of the pointer.

Parameters
atomBufferPointer to the atom data (not including the leading size and atom tag)
atomSizeSize of the atom. This is used to avoid overflow, or if the atom size was set to zero
Returns
A new, malloc'd, pointer of atom data filled with data. In case of failure, returns NULL
